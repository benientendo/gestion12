Format Simple
Colonnes attendues:
  • Code - Code article (optionnel, généré auto)
  • Nom - Nom de l'article Obligatoire
  • Prix Achat - Prix d'achat
  • Prix Vente - Prix de vente
  • Stock - Quantité en stock
  • Devise - CDF ou USD (optionnel)
Important:
  • La première ligne doit contenir les en-têtes
  • Si un code existe déjà, le stock sera ajouté
  • Devise par défaut: CDF
Exemple Simple
Code Nom Prix Achat Prix Vente Stock
ART001Savon500800100
ART002Lait1500200050
ART003Riz 5kg80001000030
Format Avancé (Articles détaillés)
Colonnes attendues:
  • Fournisseur - Nom du fournisseur (optionnel)
  • Nom de l'article - Nom de l'article Obligatoire
  • Catégorie - Catégorie de l'article
  • P.achat/u - Prix d'achat unitaire (FC)
  • Prix_ventes - Prix de vente (FC)
  • Pièces/carton - Nombre de pièces par carton
Ce modèle crée automatiquement:
  • Les articles avec leurs prix (achat et vente)
  • Les catégories si elles n'existent pas
  • Les infos fournisseur et pièces/carton en description
  • Aucun stock n'est ajouté (quantité = 0)
Important:
  • La première ligne doit contenir les en-têtes
  • Les prix sont en Francs Congolais (FC)
  • Si un article existe déjà, ses prix seront mis à jour
Exemple Avancé
Fournisseur Nom de l'article Catégorie P.achat/u Prix_ventes Pièces/carton

BrassimbaBière TurboBoissons1000130024
